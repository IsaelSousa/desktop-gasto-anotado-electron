# Desktop Gasto Anotado

## How to install
-- Install Desktop Development C++ (Visual Studio Installer).  
-- yarn or npm install  

## How to use
Execute app in Dev.  
``
    yarn start or npm run start
``  
Generate package  
``
    yarn rebuild or npm run rebuild
``  
then  
``
    yarn package or npm run package
``  
package is save in 'release' path.
